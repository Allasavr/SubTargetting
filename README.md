# Трекер подписок

Десктопное приложение для учета и отслеживания платных подписок с функцией конвертации валют. Разработано на C# в рамках учебного проекта.

## О проекте

Приложение помогает вести учет регулярных платежей за подписки (Netflix, образовательные платформы, сервисы и т.д.) и автоматически конвертирует стоимость в выбранную валюту для удобного анализа расходов.

### Возможности
- Добавление новых подписок с указанием:
  - Названия сервиса
  - Категории (медиа, образование, игры и др.)
  - Даты начала и периода оплаты
  - Стоимости в одной из валют (USD, EUR, RUB, CNY)
- Автоматическая конвертация стоимости в выбранную базовую валюту
- Просмотр истории всех платежей в виде таблицы
- Категоризация подписок для удобной статистики

## Технологии

- **Язык:** C#
- **Платформа:** .NET Framework
- **UI Framework:** WPF
- **Хранение данных:** SQLite
